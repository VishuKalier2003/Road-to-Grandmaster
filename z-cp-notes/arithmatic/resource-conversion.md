## Resource Conversion

### Trigger
When asked
- one final product needs two or more resource to create
- You start with incomplete resources and can perform certain operations to increase or change one resource to another
We need to find the minimum number of operations to create k final products

### Mistakes
- Simulating or not thinking mathematically

### Idea
- Since each operation produces a fixed quantity or a set of resource, greedy evaluation is the most feasible approach
- And we need not simulate since the operation output is fixed and deterministic as a function, we compress it into a mathematical formula

### One Liner
- Convert entire set of operations into one equation (if possible dependent upon one variable)
