## Quadratic Formula

### Trigger
When the variable to be found 
- is present or can be expressed as a quadratic function in the problem
- or the series, blocks, etc. are a triangular sequence like [1],[1,2],[1,4,6], etc.
And we need to find the variable

### Mistakes
- Trying to simulate using binary search

### Idea
- Use the positive root of the formula used to solve quadratic equations
    * x = (-b + sqrt(4ac)) / 2a

### One Liner
- Use the formula for solving quadratic equations
